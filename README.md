# Sistema de Cadastro Python

Projeto desenvolvido em Python com foco em prática de lógica de programação, estruturas de dados e construção de sistemas em terminal.

O sistema permite realizar o gerenciamento de pessoas através de um menu interativo.

---

# Funcionalidades

- Cadastrar pessoas
- Listar registros cadastrados
- Buscar pessoas por código
- Remover registros
- Validação de opções do menu

---

# Tecnologias utilizadas

- Python 3

---

# Conceitos praticados

Este projeto foi desenvolvido para praticar:

- Funções
- Listas
- Dicionários
- Estruturas de repetição
- Condicionais
- Validação de dados
- Organização de código
- Menus interativos

---

# Estrutura do sistema

O projeto utiliza:

- Lista para armazenar registros
- Dicionários para representar cada pessoa cadastrada
